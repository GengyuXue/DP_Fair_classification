# DP Fair Classification

This repository contains the code for the paper  
**Federated fairness-aware classification under differential privacy**  
https://arxiv.org/abs/2603.24392


- **CDP and FDP**  
  Contains the function code to implement: CDP-Fair and FDP-Fair.  

  This folder also includes simulation code to reproduce all numerical experiments and plots in Section 5.1.1 and Section 5.1.2.  

- **S_effect**  
  Contains the code for simulation experiments in Section 5.1.3.  

- **Real data**  
  Contains datasets and code for all results in Section 5.2.
